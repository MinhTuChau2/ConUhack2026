# Super Space Wars 3000 🚀

**Super Space Wars 3000** is a fast-paced top-down multiplayer shooter where cute aliens battle in real time. Players collect coins, upgrade their armor, and outlast their friends in chaotic space combat. The project focuses on strong gameplay systems and real-time multiplayer over visual polish.

---

## 🎮 Features

- Real-time top-down multiplayer combat  
- Coin collection and armor-based upgrades  
- In-game store with multiple armored outfits  
- In-game chat with optional text-to-speech  
- Ghost (viewer-only) mode after player elimination  

---

## 🛠 Tech Stack

- React + Vite  
- Kaplay  
- JavaScript (custom gameplay systems)  
- Node.js + Socket.IO  
- Jotai  
- ElevenLabs API
